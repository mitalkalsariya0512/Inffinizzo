# INFINIZIO CRM Dashboard

Real-time CRM dashboard connected to Google Sheets for Infinizio digital marketing agency.

## Features
- 10 dashboard sections (Overview, Employees, Tasks, Content, Social Media, Stock, Shoots, Clients, Calendar, Data Sync)
- Live Google Sheets sync
- Dark/Light mode
- Refresh button for instant data updates

## Tech Stack
- React 18 + Vite
- Tailwind CSS
- Recharts
- Google Sheets CSV API

## Deploy
Connected to Vercel for automatic deployments.
